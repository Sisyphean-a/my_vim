##  my_vim使用手册

###  基础操作

| 键位 | 功能       | 键位 | 功能       |
|------|------------|------|------------|
| s    | 保存       | Q    | 退出       |
| ff   | 打开目录   | <F8> | 打开md索引 |
| :vsp | 分割成两列 | :sp  | 分割成两行 |
| :bn  | 上一个文件 | :pn  | 下一个文件 |
| <F5> | 编译代码   |      |            |

###  markdown

| 组合键   | 功能           | 组合键 | 功能           |
|----------|----------------|--------|----------------|
| ,c       | 创建一个代码块 | ,l     | 创建一个分界线 |
| ,1,2,3,4 | 创建四种标题   | ,d     | 小型代码块     |

:table<tab> 整理列表，需要先有一个|||||

### latex

| 键位 | 功能             | 键位 | 功能         |
|------|------------------|------|--------------|
| \ll  | 编译文件         | \lv  | 查看pdf文档  |
| \li  | 查看已定义的映射 | K    | 查看宏包信息 |

###  coc

|tt|翻译单词|
|\c|打开日历|\\|打开时钟|
